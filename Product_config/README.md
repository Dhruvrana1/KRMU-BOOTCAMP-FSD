# ⚙️ Product Configuration Studio

A modern React application that allows users to customize a laptop by selecting specifications and viewing the updated price in real time.

## 🚀 Features

* 💻 Laptop Configuration
* ⚡ Dynamic Price Calculation
* 🎨 Modern UI Design
* 📱 Responsive Layout
* ✨ Smooth Animations
